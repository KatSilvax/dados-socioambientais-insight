1. PRÉ-REQUISITOS
-----------------
✅ Python 3.8 ou superior instalado
✅ Navegador web moderno (Chrome, Firefox, Edge)
✅ Conexão com internet (para carregar bibliotecas CSS/JS)

Verificar Python:
> python --version

===============================================================================

2. INSTALAÇÃO
-------------
1. Baixar/clonar o projeto
2. Abrir terminal na pasta do projeto
3. Instalar dependências:

> cd backend
> pip install flask flask-cors pandas

===============================================================================

3. INICIALIZAÇÃO
----------------
PASSO 1: Iniciar o Backend
> cd backend
> python backend.py

✅ Aguardar mensagem: "Running on http://127.0.0.1:5000"

PASSO 2: Abrir o Frontend
- Navegar até: scripts/frontend/
- Abrir arquivo: index.html (duplo clique ou arrastar para navegador)

===============================================================================

4. COMO USAR O SISTEMA
----------------------
PASSO 1: Upload do Arquivo
- Clicar em "Selecionar Arquivo CSV"
- Escolher um arquivo CSV (ex: jardim_limpo.csv)
- Botão "Analisar Dados" aparecerá

PASSO 2: Análise
- Clicar em "Analisar Dados"
- Aguardar processamento (loading aparecerá)
- Gráficos serão gerados automaticamente

PASSO 3: Visualização
- Todos os gráficos aparecerão na página
- Cada coluna do CSV vira um gráfico
- Scroll para ver todos os resultados

6. SOLUÇÃO DE PROBLEMAS
-----------------------
❌ PROBLEMA: "Erro de conexão"
✅ SOLUÇÃO: Verificar se backend está rodando em localhost:5000

❌ PROBLEMA: "Módulo não encontrado"
✅ SOLUÇÃO: Instalar dependências: pip install flask flask-cors pandas

❌ PROBLEMA: "Arquivo não carrega"
✅ SOLUÇÃO: Usar apenas arquivos CSV válidos

❌ PROBLEMA: "Gráficos não aparecem"
✅ SOLUÇÃO: Verificar conexão com internet (Chart.js é carregado online)

❌ PROBLEMA: "Interface não abre"
✅ SOLUÇÃO: Abrir index.html diretamente no navegador
